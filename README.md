# 🏏 IPL Cricket Analytics Dashboard

## 📊 Overview

This project presents an interactive **IPL Analytics Dashboard** built using Python and Power BI.
It provides insights into team performance, scoring patterns, and match dynamics across different phases of the game.

---

## 🎯 Objectives

* Analyze team-wise performance
* Understand scoring trends across overs
* Evaluate run distribution across match phases
* Identify patterns between runs and wickets

---

## 📌 Key Features

* 📈 **KPI Metrics**

  * Total Matches Played
  * Total Runs Scored
  * Total Wickets Taken
  * Average Run Rate
  * Highest Scoring Phase
  * Most Risky Phase
  * Best Efficiency Phase 

* 📊 **Visual Insights**

  * Team-wise run contribution
  * Run distribution by match phase (Powerplay, Middle, Death)
  * Runs and wickets trend across overs

* 🎛️ **Interactive Elements**

  * Filters for team and match phases
  * Dynamic visual exploration

## 💡 Key Insights

* Death overs contribute the highest scoring rate
* Middle overs show stable scoring patterns
* Wicket frequency increases in later overs
* Death overs have the highest scoring rate,
* while Middle overs contribute the highest number of wickets due to longer duration
* Middle overs are stable with moderate scoring but highest wickets taken due to most number of overs
* Powerplay has lower risk but controlled scoring
